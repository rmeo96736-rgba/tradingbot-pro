Option B — EA Auto Send (Local Server)
Iske liye computer pe Python install hona chahiye aur ek local server chalana padega.

Step 1 — Python server script (server.py):

from http.server import HTTPServer, BaseHTTPRequestHandler import json, datetime trades = [] class Handler(BaseHTTPRequestHandler): def do_POST(self): length = int(self.headers['Content-Length']) data = json.loads(self.rfile.read(length)) data['date'] = str(datetime.date.today()) data['id'] = int(datetime.datetime.now().timestamp()*1000) trades.append(data) # Save to file so dashboard can read it with open('trades.json', 'w') as f: json.dump(trades, f) self.send_response(200) self.end_headers() self.wfile.write(b'OK') def log_message(self, *args): pass print("Server chalu hai: http://localhost:8765") HTTPServer(('localhost', 8765), Handler).serve_forever()

Step 2 — Terminal mein chalao:

python server.py

Step 3 — MT5 EA mein add karo (v28 ke saath use karo):

void SendToDashboard(string sym, string tp, double pnl, double lots, string status) { string url = "http://localhost:8765/trade"; string body = "{\"sym\":\"" + sym + "\"," + "\"type\":\"" + tp + "\"," + "\"pnl\":" + DoubleToString(pnl,2) + "," + "\"lots\":" + DoubleToString(lots,2) + "," + "\"status\":\"" + status + "\"}"; char post[], result[]; StringToCharArray(body, post, 0, StringLen(body)); string hdrs = "Content-Type: application/json\r\n"; WebRequest("POST",url,hdrs,5000,post,result,hdrs); } void OnTradeTransaction( const MqlTradeTransaction &trans, const MqlTradeRequest &req, const MqlTradeResult &res) { if(trans.type==TRADE_TRANSACTION_DEAL_ADD) if(HistoryDealSelect(trans.deal)) { double profit = HistoryDealGetDouble( trans.deal, DEAL_PROFIT); double vol = HistoryDealGetDouble( trans.deal, DEAL_VOLUME); string sym = HistoryDealGetString( trans.deal, DEAL_SYMBOL); int dtype = (int)HistoryDealGetInteger( trans.deal, DEAL_TYPE); string st = profit>0?"win": profit<0?"loss":"open"; string tt = dtype==DEAL_TYPE_BUY? "buy":"sell"; SendToDashboard(sym,tt,profit,vol,st); } }

Step 4 — MT5 settings mein allow karo:

Tools > Options > Expert Advisors > Allow WebRequest for listed URL: http://localhost:8765
